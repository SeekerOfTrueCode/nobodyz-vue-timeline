# vue-nobodyz-timeline

## How to install:

### CommonJS:
```bash
npm install --save axios vue-nobodyz-timeline
```
No entry filee installation needed.

### Script:
Just add 2 scripts in order: `vue`, `vue-nobodyz-timeline` to your `document`.
